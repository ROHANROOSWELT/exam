Android Studio
UI Change App
Changes background color, text size, and bold on button click
1
Install Android Studio from developer.android.com/studio
2
Open Android Studio → New Project → Empty Views Activity → Language: Java → Finish
3
Open res/layout/activity_main.xml → switch to Code view → paste XML below
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android" android:id="@+id/layout" android:orientation="vertical" android:layout_width="match_parent" android:layout_height="match_parent" android:padding="20dp"> <TextView android:id="@+id/t" android:text="Hello World" android:textSize="20sp" android:layout_width="wrap_content" android:layout_height="wrap_content"/> <Button android:id="@+id/b" android:text="Change" android:layout_width="wrap_content" android:layout_height="wrap_content"/> </LinearLayout>
4
Open java/com.example.app/MainActivity.java → replace all content
package com.example.app; import android.graphics.*; import android.os.Bundle; import android.widget.*; import androidx.appcompat.app.AppCompatActivity; public class MainActivity extends AppCompatActivity { protected void onCreate(Bundle b){ super.onCreate(b); setContentView(R.layout.activity_main); Button btn = findViewById(R.id.b); TextView t = findViewById(R.id.t); LinearLayout l = findViewById(R.id.layout); btn.setOnClickListener(v -> { l.setBackgroundColor(Color.YELLOW); t.setTextSize(30); t.setTypeface(null, Typeface.BOLD); }); } }
5
Click ▶ Run (Shift+F10) → Select emulator or USB device → App opens, click "Change" button
If no emulator: Tools → Device Manager → Create Virtual Device → Pixel 4 → API 30

Email + Notification
Send email intent and show system notification
1
New Project → Java. In activity_main.xml, add two buttons with ids email and notify
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android" android:orientation="vertical" android:padding="20dp" android:layout_width="match_parent" android:layout_height="match_parent"> <Button android:id="@+id/email" android:text="Send Email" android:layout_width="wrap_content" android:layout_height="wrap_content"/> <Button android:id="@+id/notify" android:text="Notify" android:layout_width="wrap_content" android:layout_height="wrap_content"/> </LinearLayout>
2
Add to AndroidManifest.xml inside <application>:
<uses-permission android:name="android.permission.POST_NOTIFICATIONS"/>
3
In MainActivity.java, add imports and code:
import android.app.*; import android.content.Intent; import androidx.core.app.NotificationCompat; // In onCreate, create channel first: NotificationChannel ch = new NotificationChannel("ch","Default", NotificationManager.IMPORTANCE_DEFAULT); ((NotificationManager)getSystemService(NOTIFICATION_SERVICE)) .createNotificationChannel(ch); Button email = findViewById(R.id.email); Button notify = findViewById(R.id.notify); email.setOnClickListener(v -> { Intent i = new Intent(Intent.ACTION_SEND); i.setType("text/plain"); startActivity(i); }); notify.setOnClickListener(v -> { NotificationManager nm = (NotificationManager)getSystemService(NOTIFICATION_SERVICE); NotificationCompat.Builder nb = new NotificationCompat.Builder(this,"ch") .setContentTitle("New Message") .setContentText("Hello User") .setSmallIcon(android.R.drawable.ic_dialog_info); nm.notify(1, nb.build()); });
4
Run → Tap buttons to test
On Android 13+, grant notification permission when prompted on first run

SQLite Library App
Creates database, inserts books, reads and displays them
1
New Project → Java. In activity_main.xml, add a TextView with id text
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android" android:orientation="vertical" android:padding="20dp" android:layout_width="match_parent" android:layout_height="match_parent"> <TextView android:id="@+id/text" android:textSize="18sp" android:layout_width="wrap_content" android:layout_height="wrap_content"/> </LinearLayout>
2
In MainActivity.java inside onCreate():
import android.database.*; package com.example.myapplication; import android.os.Bundle; import android.widget.TextView; import androidx.appcompat.app.AppCompatActivity; import android.database.Cursor; import android.database.sqlite.SQLiteDatabase; public class MainActivity extends AppCompatActivity { @Override protected void onCreate(Bundle savedInstanceState) { super.onCreate(savedInstanceState); setContentView(R.layout.activity_main); // Open/Create DB SQLiteDatabase db = openOrCreateDatabase("lib", MODE_PRIVATE, null); // Create table db.execSQL("CREATE TABLE IF NOT EXISTS books(name TEXT)"); // ⚠️ FIX 1: Prevent duplicate inserts db.execSQL("DELETE FROM books"); // Insert data db.execSQL("INSERT INTO books VALUES('Java')"); db.execSQL("INSERT INTO books VALUES('Python')"); // Fetch data Cursor c = db.rawQuery("SELECT * FROM books", null); // ⚠️ FIX 2: Use StringBuilder (efficient) StringBuilder data = new StringBuilder(); while (c.moveToNext()) { data.append(c.getString(0)).append("\n"); } // ⚠️ FIX 3: Close cursor c.close(); // Display TextView t = findViewById(R.id.text); t.setText(data.toString()); } }
3
Run → Books will display on screen
No extra dependencies needed — SQLite is built into Android
Apache Cordova
Cordova Login App
Username/password login with admin/123 credentials
1
Install Node.js from nodejs.org, then install Cordova:
npm install -g cordova
2
Create and setup project:
cordova create LoginApp com.example.login LoginApp cd LoginApp cordova platform add browser
3
Open www/index.html → replace entire content:
<!DOCTYPE html> <html> <body> <h2>Login</h2> <input id="u" placeholder="Username"><br><br> <input id="p" type="password" placeholder="Password"><br><br> <button onclick="login()">Login</button> <button onclick="reset()">Reset</button> <p id="msg"></p> <script> function login(){ let u=document.getElementById("u").value; let p=document.getElementById("p").value; if(u=="admin" && p=="123") document.getElementById("msg").innerHTML="Login Success"; else document.getElementById("msg").innerHTML="Login Failed"; } function reset(){ document.getElementById("u").value=""; document.getElementById("p").value=""; } </script> </body> </html>
4
Run in browser:
cordova run browser
Username: admin | Password: 123
Apache Cordova
Cordova Geolocation
Gets and displays device latitude and longitude
1
Create Cordova project and add geolocation plugin:
cordova create GeoApp com.example.geo GeoApp cd GeoApp cordova platform add browser cordova plugin add cordova-plugin-geolocation
2
Replace www/index.html:
<!DOCTYPE html> <html> <body> <h2>My Location</h2> <button onclick="getLocation()">Get Location</button> <p id="out"></p> <script> function getLocation(){ navigator.geolocation.getCurrentPosition(pos=>{ document.getElementById("out").innerHTML = "Lat: "+pos.coords.latitude + "<br>Lon: "+pos.coords.longitude; }); } </script> </body> </html>
3
Run:
cordova run browser
Browser will ask permission to access location — click Allow

Recipe Finder App

2

ionic start RecipeApp blank --type=react cd RecipeApp
3
Open src/App.tsx → replace content:
import { useState } from 'react'; import { IonApp, IonContent, IonInput, IonButton, IonText } from '@ionic/react'; import '@ionic/react/css/core.css'; export default function App(){ const [item,setItem]=useState(''); const [res,setRes]=useState(''); return( <IonApp> <IonContent className="ion-padding"> <h2>Recipe Finder</h2> <IonInput placeholder="Enter ingredient" onIonChange={e=>setItem(e.detail.value||'')} /> <IonButton expand="full" onClick={()=>setRes(item+" Recipe")}> Find Recipe </IonButton> <IonText><h3>{res}</h3></IonText> </IonContent> </IonApp> ); }

Run in browser:
ionic serve
Opens at http://localhost:8100 automatically

BMI Calculator
Enter height (m) and weight (kg) → calculates BMI
1
Install Node.js. Then create project using Expo (easiest method):
npx create-expo-app BMIApp cd BMIApp
2
Open App.js → replace all content:
import {useState} from 'react'; import {View,Text,TextInput,Button,StyleSheet} from 'react-native'; export default function App(){ const[h,setH]=useState(''); const[w,setW]=useState(''); const[bmi,setBmi]=useState(''); return( <View style={styles.container}> <Text style={styles.title}>BMI Calculator</Text> <TextInput style={styles.input} placeholder="Height in meters (e.g. 1.75)" onChangeText={setH} keyboardType="numeric"/> <TextInput style={styles.input} placeholder="Weight in kg (e.g. 70)" onChangeText={setW} keyboardType="numeric"/> <Button title="Calculate BMI" onPress={()=>setBmi((w/(h*h)).toFixed(2))}/> <Text style={styles.result}>BMI: {bmi}</Text> </View> ); } const styles = StyleSheet.create({ container:{flex:1,padding:20,marginTop:50}, title:{fontSize:24,fontWeight:'bold',marginBottom:20}, input:{borderWidth:1,borderColor:'#ccc',padding:10, marginBottom:10,borderRadius:5}, result:{fontSize:20,marginTop:20,fontWeight:'bold'} });
3
Run:
npx expo start
4
Scan QR code with Expo Go app (Android/iOS) OR press w for web browser
Install Expo Go from Play Store/App Store for real device testing

Real-time unit converter — type KM, see Miles

import {useState} from 'react'; function App(){ const[v,setV]=useState(''); return( <div style={{padding:'20px'}}> <h2>KM to Miles Converter</h2> <input type="number" placeholder="Enter KM" onChange={e=>setV(e.target.value)} style={{padding:'8px',fontSize:'16px'}} /> <h3>Miles: {v ?
